## 01. 다중 통화를 지원하는 Money 객체

### 요구사항

- 통화가 다른 두 금액을 더하고 주어진 환율에 맞게 변환 금액 표시
- 금액(주가) * 수(주식의 수) = 결과

> 스텁 구현(stub implementation): 메서드의 서명부와 반환 명령을 작성하여 컴파일될 수 있도록 껍데기 만드는 방법

## 03. 모두를 위한 평등

### VO(Value Object)

> VO (값 객체)는 불변성을 가진 객체를 의미한다. 객체 그 자체로 의미 있는 값을 표현하고, 동일한 값을 가진 객체는 동등해야 한다.
